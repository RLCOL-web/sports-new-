# 🏆 SportsPro - Noticias Deportivas en Tiempo Real

Una plataforma moderna, colorida e interactiva estilo ESPN.

## ✨ Características

- 🎨 **Diseño Espectacular**: Colores vibrantes, gradientes y animaciones
- 📰 **Noticias Dinámicas**: Carrusel, grid y filtros
- 📊 **Resultados en Vivo**: Scores, tendencias, lo más visto
- 🎯 **Interactividad**: Modal, likes, compartir

## 🚀 Tecnologías

- HTML5
- CSS3
- JavaScript Vanilla
- Font Awesome Icons

## 📁 Archivos

- `index.html` - Estructura
- `styles.css` - Estilos
- `script.js` - Funcionalidad
- `README.md` - Documentación

## 🎮 Cómo Usar

1. Abre `index.html` en tu navegador
2. Filtra noticias por deporte
3. Usa la barra de búsqueda
4. Haz clic para leer completo

- ✅ Móvil
